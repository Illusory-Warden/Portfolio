Self-Hosted Personal Cloud Infrastructure on Raspberry Pi

A privacy-focused, cost-effective personal cloud ecosystem designed as an alternative to commercial cloud storage services.

- Achieved 54% cost savings over five years compared to commercial cloud plans
- Implemented Zero-Trust Network Access using Twingate without port forwarding
